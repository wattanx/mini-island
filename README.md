# Mini Nuxt Server Components

This repository is a minimal implementation of Nuxt Server Components

## Status

| Feature                | Status |
| ---------------------- | ------ |
| island components      | ✅     |
| async components       | ❌     |
| props                  | ✅     |
| slots                  | ✅     |

| v-for                  | ❌     |
| server only components | ❌     |
| selective hydration    | ❌     |
